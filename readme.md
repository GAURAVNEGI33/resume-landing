# ResumeFlow - Resume Builder Landing Page

A modern, responsive landing page for **ResumeFlow**, an imaginary AI-powered resume builder. This project was built by following the provided Product Requirements Document (PRD) using semantic HTML, external CSS, and a small amount of JavaScript.

## 🚀 Live Demo

https://GAURAVNEGI33.github.io/resume-landing/

## 📸 Preview

![ResumeFlow Landing Page](images/resume-landing-page-ss.png)

## ✨ Features

- Responsive design for desktop and mobile
- Semantic HTML5 structure
- Modern dark UI with a consistent accent color
- Hero section with call-to-action
- Statistics section
- Feature cards
- Resume template showcase
- Testimonials section
- Call-to-action banner
- FAQ section
- Multi-column footer
- Dynamic footer year using JavaScript (`querySelector` + `textContent`)

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
